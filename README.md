## 📖 介绍

这是一个基于 NoneBot 框架的 QQ 插件，旨在方便用户在聊天环境中快速解析哔哩哔哩（Bilibili）的各种链接。并进行评论，弹幕的解析和分析。目前还在开发中，需要进一步集成以及优化，拭目以待吧。
